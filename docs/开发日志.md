目的是构建基于React前端和FastAPI后端的AI图像检测系统，支持图像上传、点标注和目标检测功能。系统采用前后端分离架构，提供直观的用户界面和强大的后端处理能力。

## 技术栈

### 前端 (React Frontend)
- **框架**: React 19.1.1
- **构建工具**: Vite 7.1.7
- **HTTP客户端**: Axios 1.12.2
- **语言**: JavaScript + JSX
- **样式**: CSS Modules

### 后端 (FastAPI Backend)
- **框架**: FastAPI 0.118.0
- **服务器**: Uvicorn 0.37.0
- **语言**: Python 3.13
